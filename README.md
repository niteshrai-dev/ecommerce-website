# 🛒 ShopHub - Flask E-Commerce Web Application

A full-stack e-commerce web application built using Python, Flask, SQLite, HTML, CSS, and JavaScript. ShopHub provides a seamless online shopping experience with secure user authentication, product management, and shopping cart functionality.

## Features

### Customer Features

* User Registration and Login
* Secure Password Reset Functionality
* Browse and Search Products
* Add Products to Shopping Cart
* Manage Cart Items
* Place Orders

### Admin Features

* Manage Products and Inventory
* Update Stock Levels
* Manage Customer Orders
* Update Order Status

## Technologies Used

* Python
* Flask
* SQLite
* HTML5
* CSS3
* JavaScript
* Jinja2 Templates

## Project Overview

ShopHub is designed to demonstrate full-stack web development concepts, including user authentication, database management, session handling, and e-commerce workflows. The application uses Flask as the backend framework and SQLite as the database for efficient data storage and retrieval.
